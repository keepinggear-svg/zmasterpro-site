# 支先森制造邮箱使用规则

更新时间：2026-07-03

## 总原则

支先森制造的域名邮箱用于建立正式企业身份、保护账号资产、降低平台注册混乱和邮箱被滥用风险。所有邮箱都必须使用独立强密码，并用密码管理器保存。不要在项目文件、聊天记录、表格或截图里明文保存真实密码、验证码、身份证、营业执照、银行卡、付款信息。

## 邮箱分工

### `admin@zmasterpro.com`

只用于最高权限账号：

- 域名注册商
- DNS 管理
- Vercel / 服务器 / 云服务
- 备案和主体资料
- 企业邮箱管理员后台
- 支付、发票、法务、重要安全通知

规则：

- 只给本人使用。
- 不用于注册普通平台。
- 不交给自动化工具使用。
- 不公开在官网和社交平台。
- 必须开启二次验证。

### `accounts@zmasterpro.com`

用于外部平台账号：

- 百度、Bing、360、神马、搜狗等站长平台
- B2B 平台、地图平台、工具平台
- 知乎、豆瓣、小红书、公众号等内容平台的运营账号
- 需要团队留存但不是最高权限的服务账号

规则：

- 可作为平台统一注册邮箱。
- 重要平台仍需开启二次验证。
- 如果平台权限特别高，应评估是否改用 `admin@` 或至少由本人掌握二次验证。

### `ai@zmasterpro.com`

用于 AI 工具：

- OpenAI / ChatGPT / Codex
- 豆包、扣子等 AI 工具
- AI 绘图、AI 自动化、AI 内容工具
- 测试型 AI SaaS

规则：

- 不绑定域名、付款、服务器等最高权限资产。
- 不与 `admin@` 共用密码。
- AI 工具中不要保存客户隐私、身份证、营业执照、银行卡、验证码。

### `contact@zmasterpro.com`

用于对外联系：

- 官网公开联系方式
- 客户咨询
- 报价沟通
- 合作询盘
- 名片、海报、社媒主页展示

规则：

- 可以公开展示。
- 可设置转发到本人常用邮箱或微信提醒。
- 客户资料要定期归档，不要长期散落在收件箱。

## 密码与验证码规则

- 每个邮箱使用不同强密码。
- 使用密码管理器生成和保存密码。
- 不在任何文件里明文保存真实密码。
- 不把验证码发给自动化工具。
- 不把身份证、营业执照、银行卡、付款截图交给自动化流程完全处理。
- 如果必须临时协作登录，完成后立即改密码并检查登录设备。

## 管理员权限规则

- 企业邮箱管理员权限只保留给本人。
- 不给第三方工具开放管理员权限。
- 不授权不明插件读取全域邮件。
- 不把 `admin@zmasterpro.com` 作为 API、自动化、爬虫、批量注册工具的发信账号。

## 发信信誉规则

- 新域名邮箱不要刚开通就群发。
- 外发邮件主题和正文要真实，不做夸张营销标题。
- 邮件签名建议包含：支先森制造、官网、联系人、微信或电话。
- 群发客户前先做小批量测试，观察退信和垃圾箱情况。
- SPF、DKIM、DMARC 生效前，不做大批量外发。

## 平台注册建议

- 域名、DNS、服务器、备案：`admin@zmasterpro.com`
- 站长平台、B2B、地图、工具：`accounts@zmasterpro.com`
- AI 工具：`ai@zmasterpro.com`
- 官网展示、客户咨询：`contact@zmasterpro.com`

## 异常处理

出现以下情况要立即处理：

- 收到异常登录提醒
- 收到未知验证码
- 外发邮件大量退信
- 客户反馈邮件进垃圾箱
- 企业邮箱后台出现陌生成员或陌生转发规则
- DNS 中出现未知 MX、TXT、CNAME 记录

处理顺序：

1. 修改相关邮箱密码。
2. 退出所有登录设备。
3. 检查二次验证和绑定手机号。
4. 检查转发规则、收信规则、授权应用。
5. 检查 DNS 的 MX、SPF、DKIM、DMARC。
6. 必要时联系腾讯企业邮箱客服或域名服务商。
